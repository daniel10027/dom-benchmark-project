# To-Do List — Vue

Implémentation Vue 3 (Composition API, `<script setup>`, réactivité par
proxy) de l'application to-do list utilisée pour le benchmark DOM. Voir le
[README principal](../README.md) pour le contexte complet du projet.

## Lancer

```bash
npm install
npm run dev       # http://localhost:5173
```

## Build de production + benchmark

```bash
npm run build
npm run preview
```

Cliquez sur **"▶ Lancer le benchmark"** pour exécuter la suite de mesures.
